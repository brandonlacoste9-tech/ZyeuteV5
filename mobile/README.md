# Zyeute Mobile App 📱

Quebec's TikTok - React Native Mobile App

## Features

- 📱 Full-screen vertical video feed
- 🔥 Double-tap to like with fire animation
- 💬 Comments modal
- 👤 Creator profiles
- ➕ Create/Upload video
- 🔍 Search (Videos, Creators, Hashtags)
- 🔔 Notifications
- 👤 User profile
- ⚜️ Quebec leather aesthetic

## Tech Stack

- React Native
- Expo
- React Navigation
- Expo Video & Camera

## Setup

```bash
# Install dependencies
npm install

# Start development server
npm start

# Run on iOS
npm run ios

# Run on Android
npm run android
```
